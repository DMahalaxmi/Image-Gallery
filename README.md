# 📸 Image Gallery

A responsive and interactive Image Gallery built using **HTML, CSS, and JavaScript**. The gallery allows users to view images in a modern layout with a lightbox, hover effects, and image navigation.

## 🚀 Features

- Responsive image gallery
- Hover zoom effect
- Lightbox image preview
- Previous and Next navigation
- Smooth transitions
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Image Gallery
│── index.html
│── style.css
│── script.js
└── images
    ├── img1.jpg
    ├── img2.jpg
    ├── img3.jpg
    ├── img4.jpg
    ├── img5.jpg
    └── img6.jpg
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the **Live Server** extension (optional).
4. Open `index.html` with Live Server or any web browser.
